Imagem Flexível

Você está vendo uma imagem completa? Experimente ampliar o tamanho da sua janela para aproveitar a melhor experiência.

A seguir as linhas de código para obter este resultado de imagem flexível:

<picture>
<source media="(max-width: 750px)" srcset="foto-p.png" type="image/png">
<source media="(max-width: 1050px) srcset="foto-m.png" type="image/png">
<img src"foto-g.png alt="imagem flexível"
</picture>

Lembrando que sempre deve haver uma ordem do tamanho das imagens. Comece escrevendo <img> para colocar a imagem grande no site e posteriormente acrescentar as tags <source media type> em cima da tag <img src alt>com as imagens média e pequena (sendo a média em baixo da pequena).
